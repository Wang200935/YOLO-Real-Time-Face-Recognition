## ✨ 主な機能

| 機能 | 説明 |
|---|---|
| 🎯 **リアルタイム検出** | YOLOv11-Pose で顔と姿勢キーポイントを同時検出 |
| 🧠 **増分学習** | 実行中に多角度の特徴ベクトルを自動収集 |
| 🔄 **角度多様性置換** | 冗長なベクトルを自動排除し、360° カバーを確保 |
| 📸 **品質ベースのキャプチャ** | 最もシャープなフレームのみを登録 |
| 🕴️ **全身シーン対応** | 全身が映っていても顔を正確に認識 |
| 🚫 **誤登録防止** | 横顔などが見知らぬ人として重複登録されるのを防止 |
| ⚡ **マルチスレッドパイプライン** | カメラ→検出→認識→DBが独立スレッドで動作 |
| 🗄️ **SQLite WAL** | マルチスレッド対応の並行読み書き |
| 🖥️ **GUI** | モデル切替・閾値調整・アラート設定に対応 |

## 🚀 クイックスタート

### 要件

- Python **3.10+**
- USB または内蔵カメラ
- RAM：4 GB+
- OS：Windows 10+, macOS 12+, Ubuntu 20.04+, Raspberry Pi OS (64-bit)

### インストール

```bash
# 1. 仮想環境を作成
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# 2. 依存関係のインストール
pip install -r requirements.txt

# 3. 起動
python main.py
```

初回起動時に自動ダウンロード：
- `yolo11n-pose.pt`（約 6 MB）— YOLO検出モデル
- `w600k_mbf.onnx`（約 16 MB）— MobileFaceNet認識モデル

## 🎓 人物の登録方法

### カメラから（全身シーン向け）
1. **▶ 認識開始** をクリックし、カメラの前に立ちます。
2. 約 3 秒後、自動でビジターとして登録されます。
3. **➕ 人物追加** でビジター名を変更します。

### 写真から（精度重視の場合）
1. **➕ 人物追加** → 名前を入力 → 写真を選択。
2. 以下の角度で **8〜15 枚** 用意することを推奨：

| 角度 | 説明 |
|---|---|
| 正面 | カメラを直視、自然な表情 |
| 左 45° | 頭を左に約 45 度回転 |
| 右 45° | 頭を右に約 45 度回転 |
| 左 90° | 完全に左を向く |
| 右 90° | 完全に右を向く |
| やや下向き | 頭を少し下に傾ける |
| やや上向き | 頭を少し上に傾ける |
| 全身 | カメラから 2〜3 メートル離れた全身 |

## ⚙️ 設定パラメータ

`config.py` を編集して調整：

| パラメータ | デフォルト | 説明 |
|---|---|---|
| `RECOGNITION_THRESHOLD` | 0.64 | 類似度閾値（低いほど厳格） |
| `MAX_EMBEDDINGS_PER_PERSON` | 50 | 1人あたりの最大特徴ベクトル数 |
| `SKIP_FRAMES` | 2 | N フレームごとに検出（大きいほど省エネ）|
| `DETECT_CONFIDENCE` | 0.45 | YOLO 検出信頼度閾値 |
| `DETECT_IMG_SIZE` | 640 | 検出解像度（Pi は 320 推奨） |
| `DETECT_DEVICE` | "cpu" | "cuda:0" で GPU を有効化 |
| `CAMERA_INDEX` | 0 | カメラデバイスのインデックス |

## ❓ よくある質問

**カメラが開かない？**
- 他のアプリがカメラを使用していないか確認
- `config.py` の `CAMERA_INDEX` を `1` または `2` に変更してみる
- macOS：システム設定 → プライバシーとセキュリティ → カメラで許可

**モデルのダウンロードが失敗する？**
- インターネット接続を確認
- 手動で `yolo11n-pose.pt` を `data/models/` に配置
- `buffalo_sc.zip` から `w600k_mbf.onnx` を展開し `data/models/` に配置

**認識精度が低い？**
- さまざまな角度の写真を追加（8〜15 枚推奨）
- データをクリアして多角度の写真で再登録
- `RECOGNITION_THRESHOLD` を `0.55` に下げる
